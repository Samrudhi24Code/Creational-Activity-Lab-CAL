# Hello World Spring Boot Web Application

A simple Spring Boot web application that returns "Hello, World!" on the root URL.

---

## Features

- Spring Boot based REST API
- Returns a plain text greeting at `/`

---

## Technologies Used

- Java
- Spring Boot
- Spring Web

---
